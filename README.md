# Dashboard Cripto

Projeto realizando para vaga de desenvolvedor.

## Configuração

1. **Clonando o repositório:**
   ```sh
   git clone https://github.com/WBorges1984/Dashboard-Cripto.git
   
2. Instale as dependencias</br>
   ```sh
   cd Dashboard-Cripto
   ```
   ```sh
   npm install
   ```

## Executando o projeto
   ```sh
   npm run dev
   ```
